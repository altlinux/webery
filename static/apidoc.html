Git.Alt API v1

The API accepts and returns data in JSON format.

URL {schema}://{host}/api/v1/acl
  • Method
    GET
    Description
    Shows the list of supported repositories
URL {schema}://{host}/api/v1/acl/{repo}/search/{type}
  • Method
    GET
    Arguments
    repo={string}
    name of repository
    type={string}
    type of the object. Can be 'package' or 'group'
    Description
    Returns list of all objects
URL {schema}://{host}/api/v1/acl/{repo}/{type}
  • Method
    GET
    Arguments
    repo={string}
    name of repository
    type={string}
    type of the object. Can be 'package' or 'group'
    Description
    Returns list of all objects
URL {schema}://{host}/api/v1/acl/{repo}/{type}/{name}
  • Method
    GET
    Arguments
    repo={string}
    name of repository
    type={string}
    type of the object. Can be 'package' or 'group'
    name={string}
    name of object
    Description
    Shows the ACL for the specified name in the repository
URL {schema}://{host}/api/v1/search
  • Method
    GET
    Parameters
    prefix={string or NaN}
    filter objects by prefix
    limit={number or 1000}
    shows only specified number of retults
    Description
    Returns a list of tasks and subtasks
URL {schema}://{host}/api/v1/tasks
  • Method
    GET
    Parameters
    state={string or NaN}
    shows tasks with specified state
    owner={string or NaN}
    shows tasks with specified owner
    limit={number or 1000}
    shows only specified number of retults
    Description
    Returns a list of tasks
  • Method
    POST
    Description
    Creates new task
URL {schema}://{host}/api/v1/tasks/{taskid}
  • Method
    DELETE
    Arguments
    taskid={integer}
    task number
    Description
    Removes specified task
  • Method
    GET
    Arguments
    taskid={integer}
    task number
    Description
    Returns information about specified task
  • Method
    POST
    Arguments
    taskid={integer}
    task number
    Description
    Updates existing task
URL {schema}://{host}/api/v1/tasks/{taskid}/subtasks
  • Method
    GET
    Arguments
    taskid={integer}
    task number
    Parameters
    limit={number or 1000}
    shows only specified number of retults
    Description
    Returns information about specified subtask
  • Method
    POST
    Arguments
    taskid={integer}
    task number
    Description
    Creates new subtask for specified task
URL {schema}://{host}/api/v1/tasks/{taskid}/subtasks/{subtaskid}
  • Method
    DELETE
    Arguments
    taskid={integer}
    task number
    subtaskid={integer}
    subtask number
    Description
    Removes subtask
  • Method
    GET
    Arguments
    taskid={integer}
    task number
    subtaskid={integer}
    subtask number
    Description
    Creates new subtask for specified task
  • Method
    POST
    Arguments
    taskid={integer}
    task number
    subtaskid={integer}
    subtask number
    Description
    Updates subtask in task
